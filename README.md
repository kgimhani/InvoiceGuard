# InvoiceGuard 📊

InvoiceGuard is a Flask-based machine learning web application that classifies invoices into categories such as Entertainment, Office, Travel, and Utilities.

## 🚀 Features
- Invoice classification using ML model
- Web interface using Flask
- File upload support
- Category prediction dashboard

## 🛠 Tech Stack
- Python
- Flask
- TensorFlow / Scikit-learn
- HTML/CSS
- Pandas / NumPy

## 📂 Project Structure
- app.py → main Flask application
- templates/ → frontend pages
- static/ → CSS & assets
- model/ → trained ML model (not included in GitHub if large)

## ▶️ How to Run Locally

```bash
git clone https://github.com/kgimhani/InvoiceGuard.git
cd InvoiceGuard
pip install -r requirements.txt
python app.py

Then open:
http://localhost:5000
